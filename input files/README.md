# Example input files

These are **public, non-patient** annotation files for trying out SnackPGx.

| Folder | Source | n |
|--------|--------|---|
| `RM/` | Reference-material samples (e.g., GeT-RM / Coriell), targeted PGx panel format — use with `settings.xlsx` | 20 |
| `DRY WGS/` | Public whole-genome samples (ENA/SRA run accessions, e.g., `ERR…`), multi-anno format — use with `settings_WGS.xlsx` (Settings → Input Parameters) | 70 |

Load them via **File → New Project** (multi-select supported).

> Clinical/patient annotation files are **not** included in this repository and are
> excluded by `.gitignore`.
